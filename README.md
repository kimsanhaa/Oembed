<h3>oEmbed란?</h3>
<h3다른 사이트 URL을 내장된 표현을 가능하게 하는 Format입니다.
유저가 Resource를 직접 파싱하지 않고 리소스에 대한 링크를 게시할 때 웹 사이트에서 포함된 사진 또는 비디오를 표시할 수 있습니다. </h3>
